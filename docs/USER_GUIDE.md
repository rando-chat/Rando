# Rando Chat User Guide

Welcome to Rando Chat - Connect with Random People Worldwide!

## Getting Started

### Creating an Account

**Option 1: Guest Mode (Quick Start)**
1. Click "Start Chatting as Guest" on the home page
2. Your guest session lasts 24 hours
3. No email required
4. Limited to basic features

**Option 2: Create Full Account**
1. Click "Sign Up" in the top right
2. Enter your email and create a password
3. Choose a display name (1-32 characters)
4. Verify your email
5. You're ready to chat!

---

## Finding Matches

### Joining the Queue

1. Go to the Matchmaking page
2. (Optional) Add interests to improve matching:
   - Click "Add Interest"
   - Type an interest (e.g., "gaming", "music")
   - Click Add or press Enter
   - Add up to 20 interests
3. (Optional) Set age preferences
4. Click "Find Match"
5. Wait for a match (usually 15-60 seconds)

### How Matching Works

The system finds you the best match based on:
- **Shared Interests**: More overlap = better match
- **Tier Priority**: Premium users match faster
- **Wait Time**: Longer waits get priority
- **Age Preferences**: Respects your age range settings

---

## Chatting

### Starting a Conversation

1. When matched, you'll see "Match Found!"
2. You'll be redirected to the chat
3. See your partner's display name and shared interests
4. Type a message in the input box
5. Press Enter or click Send

### Chat Features

**Sending Messages:**
- Type up to 2000 characters
- Press Enter to send
- Press Shift+Enter for new line
- Messages are automatically checked for safety

**Safety Indicators:**
- Green: Message is safe
- Yellow: Message may be flagged
- Red: Message blocked (very unsafe)

**Typing Indicators:**
- See when your partner is typing
- Shown as animated dots below chat

**Message History:**
- Scroll up to see earlier messages
- Infinite scroll loads more

### Ending a Chat

1. Click the menu button (three dots)
2. Select "End Chat"
3. You'll return to the matchmaking queue
4. Chat is saved in your history (registered users only)

## Safety & Reporting

### Content Moderation

All messages are automatically checked for:
- Harassment
- Hate speech
- Spam
- Inappropriate content
- Threats

**Safety Scores:**
- 0-30%: Message blocked
- 30-60%: Flagged for review
- 60-100%: Safe

### Reporting a User

If someone violates community guidelines:

1. Click the menu in chat
2. Select "Report User"
3. Choose a category:
   - Harassment
   - Hate Speech
   - Spam
   - Inappropriate Content
   - Threats
   - Impersonation
   - Underage
   - Other
4. Write a detailed reason (10-1000 characters)
5. (Optional) Add evidence/screenshots
6. Submit report

**What Happens Next:**
- Your report is reviewed by moderators
- You can only report once every 30 minutes
- Users with 3+ reports are auto-banned
- You'll be notified of the outcome

### Blocking a User

1. Click menu in chat
2. Select "Block User"
3. You won't be matched with them again

## Troubleshooting

### Can't Find a Match?

- **Add more interests**: Increases match pool
- **Remove age restrictions**: Wider range
- **Try different times**: More users online
- **Check your tier**: Premium matches faster

### Messages Not Sending?

- **Check internet connection**
- **Verify session is active**
- **Message may be flagged**: Try rephrasing
- **Rate limit**: Wait 1 minute (max 10 msgs/min)

### Guest Session Expired?

- Guest sessions last 24 hours
- Create an account to keep your data
- Or start a new guest session

### Payment Not Working?

- **Check card details**: Verify info is correct
- **Try different card**: Some cards may decline
- **Contact support**: support@randochat.com
- **Check Stripe status**: May be temporary outage
